
# Website

## Motivations

I made this website because I really wanted to make a website. I had been lacking something to put on the `www.` subdomain of the domain I have and it would just return a `404`. This felt wrong to me so I made a website.

## Features

The website features a glitching intro, using a really boring wikipedia article on linux to make the user think that it's going to be a really boring website.

I also added a fully fledged media controller div because the song that I put in the background of my website was pretty grating and could get annoying after a while.
